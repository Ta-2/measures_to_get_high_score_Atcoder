ランダム選択

問を選択

各問題の点数を - で区切って記述してください．整数値の代わりに [a:b] と書くことで，a 点以上 b 点以下の範囲を指定することもできます． 前の問題からランダムに決めていくので，特定の状況下では適切な問題セットをご用意できないことがあります．

また，指定順にかかわらず点数の昇順に表示されます．

問題 ID

問題ページの URL 末尾の問題 ID（abc042_a のような部分）で絞り込みます．TaskScreenName と呼ばれているものです．

簡易パターン文字列では ? が任意の一文字に，* が任意の文字列にマッチします．正規表現は JavaScript の正規表現を利用します．いずれも完全一致のみが対象です．空欄でないときに有効になります．

当サイトに求められている機能ではない気もしますが，個人的に欲しくなるときがあるので追加してみました．

ABC の D 問題を見たいときに arc*_b のことも気にしなきゃいけないのが面倒なので，それ用の変更をそのうち入れたいです．
